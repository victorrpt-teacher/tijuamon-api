Welcome to Tijuamon API!

This is a simple Flask application.

Available Endpoints

  • GET /tijuamones – List all Tijuamones
  • GET /tijuamones?type=agua – Filter by type (example)
  • GET /tijuamones?name=pi – Filter by name fragment (example)
  • GET /tijuamones?min_level=30 – Filter by minimum level (example)
  • GET /tijuamones/<id> – Retrieve a single Tijuamon
  • POST /tijuamones – Create a Tijuamon
  • PUT /tijuamones/<id> – Update a Tijuamon
  • DELETE /tijuamones/<id> – Delete a Tijuamon
  • POST /batalla – Simulate battle (optional)
  • GET /estadisticas – View statistics (optional)
